# 📦 Controle de Estoque para Supermercados

Uma aplicação web desenvolvida para auxiliar no controle e na contagem de produtos em estoque, facilitando o gerenciamento de mercadorias em supermercados.

## 📖 Sobre o projeto

O **Controle de Estoque para Supermercados** foi desenvolvido com o objetivo de simular um sistema simples de gerenciamento de estoque, permitindo registrar produtos, controlar quantidades e acompanhar as informações de forma organizada.

Este projeto faz parte dos meus estudos em Desenvolvimento Front-End, aplicando conceitos de HTML, CSS e JavaScript em uma aplicação prática.

## ✨ Funcionalidades

* 📦 Cadastro de produtos
* 🔢 Controle da quantidade em estoque
* ➕ Adição de novos itens
* ➖ Atualização da quantidade de produtos
* 🗑️ Remoção de itens cadastrados
* 🔍 Visualização da lista de produtos
* 💾 Armazenamento dos dados utilizando **LocalStorage**
* 📱 Interface simples e intuitiva

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* LocalStorage
